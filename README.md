Momo Ghar – Premium Restaurant UI/UX Design

A modern UI/UX concept for an authentic Nepali restaurant, designed for both desktop and mobile platforms.

##  Overview

Momo Ghar is a restaurant website and mobile app concept that provides an elegant online food ordering experience with a modern and user-friendly interface.

##  Features

- Responsive Desktop Website
- Mobile App UI
- Modern Restaurant Interface
- Product Details Page
- Checkout Experience
- Contact & About Pages

##  Tools Used

- Figma
- Visily  (AI-assisted concept generation)
- ChatGPT (UX brainstorming & content)
